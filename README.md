# Unity MiniGames

A collection of rapid, time-boxed Unity exercises built to test and demonstrate specific C# mechanics and system architecture.

Each exercise was strictly limited to a 90-minute development window. To meet this constraint, they use primitive shapes and minimal graphics, prioritizing clean logic, code execution, and functional mechanics over visual polish.
